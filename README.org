* dotfiles
This repository is to store all my dotfiles specifically for *nix* rebuilds if the time ever comes

** Install from live iso
*** Clone
#+begin_src sh
git clone git@github.com:risolij/dotfiles.git
#+end_src


*** Mounts && Filesystems
#+begin_src sh
sudo nix --experimental-features "nix-command flakes" run github:nix-community/disko/latest -- --mode destroy,format,mount dotfiles/home/hosts/disko-config.nix
#+end_src
